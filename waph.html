Course Overview

This course covers web application programming and security, including front-end development, web APIs, and common web attacks and defenses such as XSS, SQL Injection, and CSRF.

Hands-on Projects

  • Lab 1: Foundations of the Web - setting up a web server and basic PHP
  • Lab 2: Front-end Web Development - HTML, CSS, JavaScript, jQuery, and API integration
  • Hackathon 1: XSS Attacks and Defenses - performing and defending against cross-site scripting attacks
  • Individual Project 1: Professional Profile Website - deployed on GitHub Pages with API integration
